APK 安装流程

  1. 安装方式
    • 系统程序安装
    • Android时长
    • ADB安装
    • 手机自带安装
  2. 安装过程
    • 复制APK到/data/app 目录—-> 解压文件—–>移动dex文件到/data/dalvik-cache 目录下—> 在/data/data 目录下创建应用数据目录
  3. 安装文件所在目录
    • /system/app
      系统自带应用程序，获得adb root 权限才能删除
    • /data/app
      用户程序安装目录
    • /data/data
      存放应用数据
    • /data/dalvik-cache
      apk中的dex 安装到此目录下
  4. 卸载
    • 删除安装过程中创建的所有文件及其目录

APK 目录

assert

资源文件

lib

存放so文件

META_-INFO

apk 签名文件

  • CERT.RSA
    签名证书文件里面存放着公钥和加密算法的描述
  • CERT.SF
    存放着的是加密文件使用的私钥对摘要明文加密后得到的密文信息，只有使用秘钥配对的公钥才能解密该文件
  • MANIFEST.MF
    文件摘要，存放的是程序清单文件，包含包中所有文件的摘要明文

    res(经过aapt编译后的文件)

  • drawable
    存放图片和xml文件
  • layout
    布局文件
  • menu
    菜单文件
  • mipmap
    图片资源

androidmanifest.xml

配置文件包含组件的配置和声明

adb使用

  • 显示当前连接设备
  • 安装apk
  • 获取当前活动界面信息
  • 获取包信息
  • 获取内存信息
  • 获取数据库文件
  • 获取已安装的程序列表
  • 获取指定程序所在的路径
