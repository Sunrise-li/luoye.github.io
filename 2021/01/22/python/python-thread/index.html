线程

线程池

进程池

ProcessPoolExecutor 是 Executor 的子类，它使用进程池来实现异步执行调用。 ProcessPoolExecutor 使用 multiprocessing 回避 Global Interpreter Lock 但也意味着只可以处理和返回可序列化的对象。

协程

  • event_loop
    相当于一个无限循环，将函数注册到这个事件上，当满足条件时，函数会被执行

  • coroutine
    协程对象，将协程对象注册到事件循环中，他会被事件循环调用,可以使用async 关键字定义一个方法
    这个方法调用时不会立即执行，而是返回一个协程对象

  • task
    任务，是对协程对象的进一步封装，包含了任务的各种状态

  • future
    代表将来执行或还没被执行的任务，和task本质上没有区别

  • async
    定义一个协程

  • await
    用来挂起阻塞方法的执行

  • 使用

  • 协程注意事项
    在异步协程中如果出现同步代码，那么久无法实现异步
    当在asyncio中遇到阻塞操作时 必须使用await 手动挂起

  • aiohttp
    requests 是同步的不能在协程中使用，aiohttp是基于异步的
